# Launching WAHO Solution

To launch the WAHO solution, follow these steps:

1. Open your web browser (Edge, Chrome, Mozilla, etc.).
2. Type the following URL in the address bar: https://waho-essmed.org/
3. Press Enter. This will take you to the WAHO portal homepage.

That's it! You should now be able to access the WAHO solution.
